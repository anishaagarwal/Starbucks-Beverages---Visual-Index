What's in your
STARBUCKS beverage?

I am an avid coffee lover, who is always on the lookout for trying new coffee blends and beverages. Starbucks, the multinational coffee chain provides a plethora of beverages, each with a varying combination of ingredients. Using the data set from Kaggle I've created a visual index of the nutritional facts of the various beverages in Starbucks.

Each visual index represents a beverage at starbucks. I have considered the data for tall size drinks and non-fat milk type for this visualization.

4 data points have been mapped for every beverage, size corresponding to each of which represents the value of that data point. Caffeine and Calorie content have been plotted using a line, the lenghth of which is directly proportional to the quantitative value. Similarly, circles have been used to represent total carbs and sugars in each beverage. The radius of each circle represents the amount of each value.

  • Caffeine Content (0-300mg)
  • Number of Calories (0-300)
  • Total Carbs (0-190g)
  • Sugar Content (0-50g)
